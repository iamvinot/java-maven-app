### Prerequisites
 
 * Staging/Production Environment - EC2 instance(s) with Kubernetes running
 * CI/CD Automation Server - EC2 instance with Jenkins Setup
 * Dev Environment - Any Machine with an IDE and Git
 * Notification - Slack
 * Image Repository - ECR
